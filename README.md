### 🜁 About the Project

**Gold AI Prediction** adalah aplikasi mobile yang menggunakan **Artificial Intelligence** untuk memprediksi harga Emas (XAUUSD) secara akurat. Proyek ini menggabungkan model machine learning (Python) dengan antarmuka mobile yang modern dan responsif.

Dibangun dari Jakarta untuk trader, investor, dan analis pasar yang membutuhkan insight cepat dan akurat tentang pergerakan harga emas.

### ✨ Key Features

- **AI Prediction Engine** — Model machine learning untuk memprediksi harga emas harian/mingguan
- **Real-time Market Data** — Integrasi data live XAUUSD
- **Mobile-First Interface** — Aplikasi mobile yang cepat dan user-friendly (Android & iOS)
- **Interactive Dashboard** — Grafik harga, tren, dan probabilitas prediksi
- **Model Training & Evaluation** — Notebook Python untuk training ulang model
- **Dark Mode** — Tampilan elegan dengan tema gelap (sesuai vibe proyek)
- **Export Report** — Hasil prediksi dapat diekspor dalam format PDF/CSV

### 🚀 Installation & Usage

```bash
# Clone repository
git clone https://github.com/Gharudha/Gold-AI-Prediction.git

# Masuk ke folder
cd Gold-AI-Prediction

# Install dependencies mobile app (Expo)
npm install
# atau
yarn install

# Jalankan aplikasi mobile
npx expo start

# Untuk bagian AI (Python)
cd server
pip install -r requirements.txt
python main.py
